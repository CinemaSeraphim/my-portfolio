Industrial Project | Server & Website Management | 2024

An ITE campus-led project where we learn to create and manage a web server alongside develop skills and knowledge for back-end web development

Due to the nature of the assignment, students were required to be extremely well-verse in the topics and subjects of HTML programming, web development and database management. In addition, students needed to know how to famaliarise themselves with navigating Visual Studio (Microsoft Application) and Database Mangement 2023.
